Functions

A function is a block of code which only runs when it is called. You can pass data, known as parameters, into a function. A function can return data as a result.

Functions serve 2 main purposes:

  • Create self-contained, reusable modules (Decomposition)
  • Suppress detail (Abstraction)

Examples for function definition and invocation:

    def greet( name):
	print( "Hello", name)
	
greet( 'John')

    Average of list elements

    def avg( lst):
	sum = 0
	for el in lst:
		sum += el
	return sum/len(lst)
	
print( avg( [12, 34, 65]))

    Square root of a number

    Babylonian method: rootn+1 = (rootn + N/rootn)/2

    def sq_root( N, eps):
	print( 'Approximate square root of {} within +/- {}'.format( N, eps))
	root= N/2
	while abs( root**2 - N) > eps:
		print( '\tcurrent root', root)
		root= ( root + N/root)/2
	return root

print( sq_root( 34.0, 0.001)

    Sort a list using Bubble sort

    def bubble_sort( lst):
	for last in range( len(lst)-1, 0, -1):
		for i in range( last):
			if lst[i] > lst[i+1]:
				lst[i], lst[i+1] = lst[i+1], lst[i]	# swap
	return lst

print( bubble_sort( [34.0, -56, 0.001]))
print( 'Better sort:', sorted( [34.0, -56, 0.001]))
