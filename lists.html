Lists

The variable types we saw so far contain a single value (a number or a string).

Many problems require dealing with collections of data. Python supports several types of collection (array, tuple, dictionary, set, and class). We will look at lists, which are most common.

Working with Lists

A list is an ordered sequence of values.

myList = [12.5, 4, 102, -35]
children = ['Heiko', 'Nina']

Individual elements of a list can be accessed using an 'index' starting at zero. The last index is -1.

myList[0] = 11
print( '2nd:', myList[1])
print( 'last:', myList[-1])

List operations include: +, insert, append, sort, and many more.

To iterate over all values of a list we use the 'for' loop.

for element in myList:
	print( '\t', element)

Exercises

myList = [ 'spam', 'eggs', 'rice']

Find all divisors of a number 

num = int( input( 'enter a number: '))
divisors = []
for i in range( 1, num):
	if num % i == 0:
		divisors.append( i)
print( 'Divisors:', divisors)

Create a string consisting of list elements separated by '_' 

str = ''	# initialize the string
for el in myList:
	str += el + '_'
str = str[0:-1]	# remove the trailing underscore
print( 'Combined:', str)

print( 'Better combine:', "_".join( myList))

Reverse the order of elements 

rev = []
for index in range( len(myList), 0, -1):
	rev.append( myList[index-1])
print( 'upsideDown:', rev)

myList.reverse()
print( 'Better reverse:', myList)
