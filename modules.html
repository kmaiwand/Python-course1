Python Modules

A module is a file containing Python definitions and statements. The file name is the module name with the suffix '.py' appended.

User Modules

If you quit from the Python interpreter and enter it again, the definitions you have made (functions and variables) are lost. Therefore, if you want to write a somewhat longer program, you are better off using the IDLE editor or a text editor to prepare the input for the interpreter and running it with that file as input instead.

Explore: modules docs.python.org.

System Modules

Python comes with an extensive library of standard modules. These extend the core language and can be platform dependent.

Below is an example for using a standard library

import random
def gamble():
	""" roll 2 dice """
	min, max = 1, 6
	roll_again = "y"
	while roll_again == "y":
		print( "Rolling the dice...", random.randint(min, max), ',', random.randint(min, max))
		roll_again = input("Roll the dice again (y/n)?")

gamble()
