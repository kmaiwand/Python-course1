Conditions

A condition is an expression that is either true or false.

var1 == var2
num1 > num2

Below are usage examples for conditional statements:

1. Perform an action only if a condition is true.

if condition:
	print( 'True')

2. Based on a condition perform one action or another.

num = int( input( 'Enter a number: '))
if( num % 2 == 0):
	print('Even')
else:
	print('Odd')
	if num % 3 != 0:
		print( 'And not divisible by 3')

Note:

  • Either branch can have multiple statements.
  • The actions in either branch have to be indented! Indentation is Python's way of grouping actions. Statements at the same level of indentation belong together.
